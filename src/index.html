H.E.S.S. Galactic Plane Survey
HGPS paper (AA/2017/32098) online material

H.E.S.S. is an array of ground-based gamma-ray telescopes located in Namibia. The H.E.S.S. Galactic Plane Survey (HGPS) is the first deep and wide survey of the Milky Way in TeV gamma-rays.

This webpage contains the online material (survey maps, source catalog, figures) for the HGPS paper AA/2017/32098 that was accepted for publication in A&A in 2017. We will link to the paper on arXiv and A&A as soon is available. This paper and the data products below cover HGPS observations taken from 2004 to 2013.

If you have any questions or commments, contact the corresponding authors: Christoph Deil, Axel Donath, Ryan Chaves, Francois Brun, Vincent Marandon. The general contact address for H.E.S.S. is contact.hess@hess-experiment.eu.

Extras

This section contains some information that you might find helpful if you want to browse or work with the HGPS data.

  • The FITS files available for download can be viewed with any FITS viewer. For the survey maps you can use e.g. DS9 or Aladin, for the catalog and other tables TOPCAT.
  • To work with the images (e.g. look up the value at a given position or cut out a small image for a given source), or with the catalog table (e.g. look up the spectrum for a given source, or convert to some other format like a DS9 region file), one option is to use Python. We have prepared a Jupyter notebook that shows some examples how to work with the HGPS data using Python, Astropy and Gammapy. See hgps_notebook_v1.html or download hgps_notebook_v1.ipynb to execute the notebook locally.
  • The HGPS survey maps and catalogs are being added to Vizier at CDS. Production of a HIPS version of the HGPS maps is planned, which can be viewed using e.g. Aladin Desktop or Aladin Lite without having to download the FITS images first. We will add links here as soon as those are available.
  • We plan to add the HGPS map to gamma-sky.net as soon as the HIPS version is available. The HGPS catalog data will be added to gamma-cat. This will allow you to easily compare the HGPS data to other gamma-ray and multi-wavelength data.
  • If there are other webpages or tools to browse or work with HGPS data, please let us know and we'll add them here.
